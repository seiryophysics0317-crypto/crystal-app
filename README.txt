crystal-app v5

修正内容
1. 画面手前から常にライトが当たるように、カメラ追従ライトを追加しました。
   どの角度に回転しても、手前側が暗くなりにくくなっています。
2. 右上に「角度設定」ボタンを追加しました。
   ・初期：一番見栄えがよいおすすめ角度
   ・斜め上：別方向からの斜め表示
   ・正面：正面からの観察
   ・横：横からの観察
   ・真上：上からの観察
3. 角度設定ボタンを押すと、現在表示している構造の大きさに合わせて、カメラ距離も自動調整します。
4. v4までの修正も含んでいます。
   ・設定パネルを除いた右側画面の中央に構造を表示
   ・HCP結晶いっぱい版を立方体状ブロック表示
   ・HCP 1/3カット
   ・イオン結晶の半径比・接触表示倍率

実行方法
npm install
npm run dev

ブラウザで表示されたURLを開いてください。

本番用ビルド
npm run build

GitHub Pages等で公開する場合は dist フォルダを使います。
